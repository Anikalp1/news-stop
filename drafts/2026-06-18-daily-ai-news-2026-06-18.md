# Agents Grow Smarter, Satellites See Vision‑Language, and Attention Turns Linear  

AWS expands its Bedrock AgentCore, India wrestles with Anthropic’s model pause, a zero‑shot vision‑language model flies in orbit, researchers probe long‑term agent planning, and a new attention variant promises linear‑time scaling. The week’s releases push both the capabilities of autonomous software and the hardware that feeds it.

## New in Amazon Bedrock AgentCore: Build agents with broader knowledge and continuous learning  

**What happened:**  
Amazon Web Services announced updates to Bedrock AgentCore that let developers create agents with expanded knowledge bases and ongoing learning loops.  

**Why it matters:**  
Developers can now ship agents that stay current without redeploying models, reducing maintenance overhead. The broader knowledge integration simplifies building multi‑domain assistants that adapt to new data sources on the fly.  

## As Anthropic suspends access to new models, India debates its AI future  

**What happened:**  
Anthropic halted rollout of its latest models, prompting a public discussion in India about the nation’s AI strategy.  

**Why it matters:**  
Startups relying on Anthropic’s APIs must pivot to alternative providers or self‑hosted solutions, accelerating interest in open‑source or local models. The policy debate may shape future regulatory frameworks that affect model licensing and data sovereignty.  

## NAVI-Orbital: First In‑Orbit Demonstration of a Zero‑Shot Vision‑Language Model for Autonomous Earth Observation  

**What happened:**  
A low‑Earth‑orbit spacecraft deployed NAVI‑Orbital, a software system that performed zero‑shot vision‑language tasks on Earth‑observation data without ground‑side processing.  

**Why it matters:**  
Developers can now envision pipelines where raw sensor data is interpreted onboard, cutting down downlink bandwidth and latency. The demo opens pathways for real‑time analytics in remote sensing, disaster response, and agricultural monitoring.  

## CEO‑Bench: Can Agents Play the Long Game?  

**What happened:**  
A new arXiv paper evaluates language‑model agents on long‑horizon tasks, noisy information acquisition, and uncertainty navigation.  

**Why it matters:**  
The study highlights gaps in current agent toolkits, nudging builders to incorporate planning modules, memory structures, and robust retrieval. Teams aiming for autonomous workflows—like end‑to‑end code generation or multi‑step customer support—should watch for emerging frameworks that address these challenges.  

## Gaussian Mixture Attention: Linear‑Time Sequence Mixing via Probabilistic Latent Routing  

**What happened:**  
Researchers introduced Gaussian Mixture Attention, replacing pairwise query‑key comparisons with a learned routing mechanism that runs in linear time.  

**Why it matters:**  
Linear‑time attention makes it feasible to feed transformers much longer contexts without exploding compute costs. Developers working on large‑scale language or multimodal models can experiment with GMA to lower training budgets and enable richer prompt windows.
